No config changes. Pipeline-stage awareness added to the `ai-lore` master skill (brainstorm and architect sessions now appear in the state menu), and completed/merged plan and brainstorm folders are now archived to `.ai-lore/complete/` automatically by `ail-cleanup`. A manual "Archive abandoned item" menu option is also available. Update `plugin_version` to `0.14.0`.

## One-time migration steps

1. Read `.ai-lore/runs.yaml`. If it does not exist or has no runs, skip to step 4.
2. For each run entry with `status: merged`:
   - If `.ai-lore/plans/<slug>/` still exists (not yet archived):
     ```
     mkdir -p .ai-lore/complete/plans
     mv .ai-lore/plans/<slug>/ .ai-lore/complete/plans/<slug>/
     ```
   - If `.ai-lore/brainstorm/<slug>/` exists:
     ```
     mkdir -p .ai-lore/complete/brainstorm
     mv .ai-lore/brainstorm/<slug>/ .ai-lore/complete/brainstorm/<slug>/
     ```
3. Report each folder moved, or "nothing to migrate" if none were found.
4. Update `plugin_version: 0.14.0` in config.yaml.
